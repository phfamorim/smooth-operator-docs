# Smooth Operator — Documentation

Documentation site for [Smooth Operator](https://claude.ai/code/artifact/3bfe686f-1d83-46aa-8c77-07c0e7b085a1), the SDF voxel remesher for Blender 5.1+.

The whole site is one self-contained `index.html` — no build step, no dependencies.

## Updating

Edit `index.html`, commit, push. GitHub Pages redeploys automatically within a minute or two.

The page's changelog section should mirror `CHANGELOG.md` in the addon package with every release.
